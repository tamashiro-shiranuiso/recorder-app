# 社内会議レコーダー (フロントエンドUI)

このリポジトリは、社内向けの音声録音ツールの「操作画面（見た目）」のみを公開・ホスティングしている場所です。

### 🔒 セキュリティと機密保持について
- **音声データの不在**: このリポジトリ（およびGitHubのサーバー内）には、職員の個人情報、会議内容、録音された音声データなどは一切保存されません。
- **データの通信ルート**: 録音された音声データは、通信が全て暗号化（HTTPS）された状態で、利用者が各自で設定した個人のGoogleドライブ（Google Apps Script）へとダイレクトに送信されます。インターネット上にデータが公開・蓄積されることはありません。
